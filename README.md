# Weiyi Xia Personal Website

Static GitHub Pages personal homepage for `weiyixia.github.io`.

## Local Preview

Open `index.html` directly in a browser, or run a tiny static server from this directory:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Publishing

The site is designed to be published from the repository root on the `main` branch:

```bash
git push origin main
```
